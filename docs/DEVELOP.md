# DEVELOP

## API DEVELOP

### Steps

---

1. Navigate to `backend-blip` directory.
2. Install necessary packages using `package.json`.
3. Start the development server.

### Code

---

```
cd backend-blip
npm install
npm run dev
```

## FRONTEND DEVELOP

### STEPS

---

1. Navigate to `frontend-blip` directory.
2. Install necessary packages using `package.json`.
3. Start the development server.

### Code

---

```
cd frontend-blip
npm install

```
